About TaskMaster

This is a simple To-Do application built to practice Flask routing, Jinja2 loops, and template inheritance.

Version: 1.0.0

Project Details

Framework Flask (Python)
Used as the core backend engine to handle URL routing and server-side logic.
Templating Jinja2
Enables dynamic content rendering and template inheritance for a consistent UI.
Styling Custom CSS3 (Grid & Flexbox)
Powers the responsive task dashboard and modern card-based layout design.
Data Handling In-Memory Storage
Uses Python dictionaries and lists to manage task data efficiently in real-time.
